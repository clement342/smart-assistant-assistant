# Smart Text Assistant (Python)

A beginner Python project that analyzes text input and gives simple automated responses.

---

## 🚀 What it does

- Detects mood in text (positive, negative, neutral)
- Identifies if a sentence is a question or a statement
- Gives a simple automated response based on input

---

## 🧠 What I learned

- String manipulation in Python
- Using loops and conditions
- Basic logic for decision making
- Introduction to automation thinking
